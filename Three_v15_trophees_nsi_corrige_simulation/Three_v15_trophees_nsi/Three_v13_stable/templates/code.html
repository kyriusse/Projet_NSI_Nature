Page Code - Frank SQL

Cette page permet de piloter rapidement le projet avec le mini-langage Frank SQL. Les autres ajouts rapides restent specifiques a leur page, tandis qu'ici on travaille directement avec le langage.

Editeur Frank SQL

Bibliotheque Frank SQL

Creer une table

tab{
Economie(prix=float;stock=int;etat=texte)
}

Ajouter un objet

obj{
Usine(prix=120,stock=50);
Client(etat=commande)
}

Ajouter une colonne

col{
prix_moyen=float;
stock=int
}

Creer des composants

cp{
Smartphone(Batterie,Ecran,Coque)
}

Ajouter une position

map{
Usine(x=152,y=87);
Port(x=58,y=43)
}

Selection de simulation

ch{
Livraison(Usine,px(58;87),Client)
}

Evenement

evt{Commande(obj=Client ; =commande ; obj=Colis ; op=+1 ; 1)}

Paterne

pat{Production(obj=Usine ; op=+5 ; 2)}
