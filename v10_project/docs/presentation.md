# Presentation

Le site reste volontairement simple. Le coeur du projet est en Python :

- gestion d'une base SQLite
- creation de graphes a partir des liaisons
- parcours BFS et DFS
- simulation par tours
- propagation d'un effet dans un reseau
- conversions entre unites

Le projet essaye d'utiliser des structures et des algorithmes proches du programme de NSI.
