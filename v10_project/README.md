# Projet NSI V10

Projet Flask + SQLite pense pour les Trophees NSI.

## Lancer le projet

```bash
cd sources
pip install -r ../requirements.txt
python app.py
```

Puis ouvrir http://127.0.0.1:5000

## Idees NSI mises en valeur

- base de donnees SQLite
- listes et dictionnaires
- liste d'adjacence
- matrice d'adjacence
- parcours BFS et DFS
- recherche de chemin
- simulation par tours
- propagation dans un graphe
- conversions d'unites par parcours dans un graphe d'unites
