**Tree :**

Gloablement => Tree est une application web développée en Python avec Flask permettant de créer et simuler des données. 

C'est une sorte de moteur de simulation. Il possède son propre langage, le Frank SQL, permettant de décrire les simulations 
(automatisation de la création de structures et de simulations)

Plus précisemment, on créer des objets et on regarde leurs évolutions dans le temps à travers une courbe et un journal de bord. 
Le projet permet notamment de modéliser des systèmes complexes sous forme de graphes (économie simplifiée, écosystèmes, réseaux d’influence, etc.)
et d’observer les interactions entre les différents éléments grâce à une simulation.
En effet, nous pouvons lier des objets. Créer des réseaux. Créer des événements et des paternes (événements à une fréquence n).
Nous, nous voulions lier nature et économie et montrer les impacts entre c'est deux choses. 

Le langage Frank SQL que nous avons inventés, à pour vocation de permettre aux utilisateurs de partager leur simulations.

Il existe 10 000 façons d'utiliser Tree.

**Conseil d'utilisation**

Le projet est parfois très dense, 
Si vous avez un oubli, ou une question, et que vous êtes déjà sur le site, 
la page Help du projet peut être très utile car elle contient nottament la bibliothèque Frank SQL permettant de comprendre le code.

Avant de vous lancer dans le projet, je vous recommande de lire attentivement le fichier *mode_d_emploie* dans le dossier docs.
Aussi, il est conseillé de voir les test et exemple d'utilisations.

Enfin, nous avons rendu l'utilisation du Frank SQL non nécessaire, vous pouvez donc rentrer les données à la mains. Nous avons quand même mis à votre disposition des 
"Ajout rapide" permettant d'allez quand même plus vite. 


**Pré-requis :**
- Python 3.10 minimum
- la bibliothèque Flask. (vous pouvez executer lancer.py pour vérifier que vous l'avez bien)
installation Flask => pip install flask
- Navigateur web (Chrome, Firefox, Edge)
- bibliothèque : flask, sqlite3, os, math, random, html, re, sys, collections

**installation**

vous pouvez executer lancer.py pour vérifier que vous avez bien Flask et l'installer si non.
Si vous n'avez pas flask et que lancer.py ne fonctionne pas :
rentrer dans un terminale : pip install flask


**Lancer :**
Il suffit d'excuter le fichier python : app.py
ensuite il faut rentrer cette adresse dans le navigateur : http://127.0.0.1:5000
